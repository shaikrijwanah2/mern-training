# 🌐 Web Fundamentals — HTML Reference

A complete beginner-to-intermediate HTML reference folder, organized by topic.  
Each file is a standalone, well-commented reference you can open in any browser.

---

## 📁 File Structure

```
web-fundamentals/
│
├── 📄 how-the-web-works.txt           → Theory: Browser, Server, HTTP, DNS
├── 📄 tags-and-attributes.txt         → Theory: HTML tags, attributes, id/class
│
├── 🌐 html-structure.html             → HTML skeleton, head, body, comments
├── 🌐 document-structure.html         → header, nav, main, section, article, aside, footer
├── 🌐 headings-paragraphs.html        → h1–h6, p, br, hr, text formatting
├── 🌐 lists.html                      → ul, ol, dl, nested lists
├── 🌐 links.html                      → anchor tag, href, target, mailto, download
├── 🌐 images.html                     → img tag, alt, src, figure, lazy loading
├── 🌐 tables.html                     → table, thead/tbody/tfoot, colspan, rowspan
├── 🌐 semantic-tags.html              → All HTML5 semantic elements explained
├── 🌐 form-elements.html              → form, input, textarea, select, checkbox, radio
├── 🌐 input-types.html                → All 20+ HTML input types with live demo
├── 🌐 form-validation-basics.html     → required, pattern, min/max, CSS validation
└── 🌐 student-registration-form.html  → 🔥 Full practice project
```

---

## 🚀 How to Use

1. Clone or download this repository
2. Open any `.html` file directly in your browser
3. Read the `.txt` files in any text editor
4. Use the files as quick-reference guides while learning

```bash
git clone https://github.com/your-username/web-fundamentals.git
cd web-fundamentals
```

## ✨ Features

- Every HTML file is **self-contained** (styles included, no external dependencies)
- Theory files are **cleanly formatted** `.txt` with ASCII diagrams and tables
- HTML files include **live demos** so you can see output directly in the browser
- The **Student Registration Form** covers: all input types, validation, semantic structure, and accessibility best practices
- All files are **beginner-friendly** with detailed comments
