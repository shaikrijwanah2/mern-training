Destructuring Assignment

Easily extract values from arrays or objects.

Array destructuring:
const colors = ['red', 'green', 'blue'];
const [first, second] = colors;
console.log(first);  // 'red'
console.log(second); // 'green'

// Skip items
const [,, third] = colors;
console.log(third); // 'blue'

Object destructuring:
const person = { name: 'Alice', age: 30, city: 'NYC' };
const { name, age } = person;
console.log(name); // 'Alice'
console.log(age);  // 30

// Rename variables
const { name: fullName, city: location } = person;
console.log(fullName); // 'Alice'
console.log(location); // 'NYC'

Default values:
const { country = 'USA' } = person;
console.log(country); // 'USA' (since person.country is undefined)

Nested destructuring:
const user = { id: 1, profile: { username: 'jsmith' } };
const { profile: { username } } = user;
console.log(username); // 'jsmith'
